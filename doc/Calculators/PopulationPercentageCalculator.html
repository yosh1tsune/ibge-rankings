Overview

Serviço responsável por calcular o percentual de frequência de ocorrência em relação à população residente de uma localidade, com base em dados do IBGE.

Examples:

service = Calculators::PopulationPercentageCalculator.new(
  locality_id: '3550308', frequency: 10000
)
service.execute # => 5.27
